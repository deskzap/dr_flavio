# Dr. Flavio - Oftalmologista (Website Dinâmico)

Projeto de desenvolvimento do novo website do oftalmologista Dr. Flávio.
O objetivo é criar um portfólio web de oftalmologia extremamente moderno e dinâmico a partir do zero.

## Arquitetura e Tecnologias
- **Backend:** PHP
- **Banco de Dados:** MySQL / MariaDB
- **Frontend:** HTML, CSS, JavaScript (Foco em design de alta qualidade, premium e responsivo)
- **Servidor Local (Dev):** WampServer (Windows)
- **Servidor de Produção:** Linux, Docker (Case-sensitive)

## Funcionalidades Principais
O site será totalmente gerenciável, possuindo as seguintes interfaces:

1. **Frontend (Área Pública):**
   - Apresentação profissional dos serviços do Dr. Flávio.
   - Design atualizado, focado na experiência do usuário (UX/UI).
   - Informações dinâmicas alimentadas pelo banco de dados.

2. **Backend (Painel Administrativo):**
   - Sistema de Autenticação (Login) restrito ao proprietário/administradores.
   - CRUD para controle total do conteúdo do site:
     - Menus de navegação
     - Postagens (Blog/Notícias)
     - Gestão de Imagens/Banners
     - Informações de contato (Telefones, Endereços)
     - Datas e Nomes da equipe/serviços
